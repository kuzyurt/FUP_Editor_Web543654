# FUP Editor Web

## Overview
The FUP (FUP=FBF=Function Block Diagram) SPS / IEC 61131‑3 Editor Web project is an web-based application designed for editing and managing features of the FUP (Feature Update Protocol). This documentation provides insights into the setup instructions.


## Setup Instructions
1. **Prerequisites:** Ensure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your machine.
2. **Clone the repository:**
   ```bash
   git clone https://github.com/kuzyurt/FUP_Editor_Web543654.git
   cd FUP_Editor_Web543654
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the application:**
   ```bash
   npm start
   ```
5. **Access the application:** Open your browser and go to `http://localhost:3000`.
